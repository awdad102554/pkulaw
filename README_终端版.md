# 北大法宝图书馆自动爬虫 - 终端版

天津大学图书馆专用版本，无需GUI界面，直接在终端运行。

## 文件说明

| 文件 | 说明 |
|------|------|
| `北大法宝图书馆自动爬虫_终端版.py` | **主程序** - 交互式命令行界面 |
| `run_auto.py` | 一键运行脚本 - 使用config.py配置直接执行 |
| `config.py` | 配置文件 - 设置账号密码和搜索参数 |
| `一键运行_全自动.bat` | Windows批处理，双击直接运行 |
| `启动Chrome调试模式.bat` | 启动Chrome调试模式的批处理 |

## 快速开始

### 方法一：交互式运行（推荐首次使用）

1. **启动Chrome调试模式**
   ```bash
   # 双击运行：启动Chrome调试模式.bat
   # 或者手动运行：
   chrome.exe --remote-debugging-port=9333
   ```

2. **运行爬虫**
   ```bash
   python 北大法宝图书馆自动爬虫_终端版.py
   ```

3. **按提示操作**
   - 选择操作模式（输入数字）
   - 输入搜索参数
   - 等待程序自动完成

### 方法二：一键运行（配置好后）

1. **修改配置文件**
   打开 `config.py`，设置搜索参数：
   ```python
   DEFAULT_KEYWORD = '合同纠纷'  # 搜索关键词
   DEFAULT_CASE_TYPE = '民事案件'  # 案件类型
   DEFAULT_YEAR = '2024'  # 年份
   DEFAULT_MAX_PAGES = 10  # 翻页数
   ```

2. **双击运行**
   ```
   一键运行_全自动.bat
   ```

## 注意事项

1. **必须先启动Chrome调试模式**
   ```bash
   chrome.exe --remote-debugging-port=9333
   ```

2. **首次使用建议先测试各步骤**
   - 运行主程序选择手动操作（选项2-6）
   - 确认每一步都能正常工作

3. **如遇页面结构变化**
   - 程序会生成调试HTML文件
   - 根据页面结构调整元素选择器

4. **网络问题**
   - 程序会自动检测网络超时
   - 遇到致命错误会中断并保存进度

## 故障排查

### 无法连接浏览器
```
[✗ ERROR] 连接浏览器失败
[⚠ WARNING] 请确保Chrome已启动调试模式
```
**解决**: 运行 `启动Chrome调试模式.bat` 或手动启动Chrome调试

### 登录失败
```
[✗ ERROR] 登录失败，请检查账号密码
```
**解决**: 检查config.py中的账号密码是否正确

### 未找到元素
```
[⚠ WARNING] 未找到列表元素
```
**解决**: 页面结构可能已更新，查看debug_*.html文件分析页面结构

## 技术架构

- **浏览器控制**: DrissionPage (Chromium)
- **Chrome端口**: 9333 (可配置)
- **登录方式**: 图书馆统一认证 → 北大法宝
- **数据存储**: 本地txt文件
- **防封策略**: 随机等待2-5秒
