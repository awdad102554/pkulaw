# 北大法宝图书馆自动爬虫 - 使用说明

## 文件说明

| 文件 | 说明 |
|------|------|
| `北大法宝图书馆自动爬虫.py` | **主程序** |
| `启动Chrome调试模式.bat` | 启动Chrome调试模式 |
| `build_exe.py` | 打包脚本（可选） |

## 使用步骤

### 第一步：启动Chrome调试模式

双击运行 `启动Chrome调试模式.bat`，或者手动执行：
```bash
chrome.exe --remote-debugging-port=9333
```

### 第二步：运行爬虫

```bash
python 北大法宝图书馆自动爬虫.py
```

### 第三步：输入参数

```
搜索关键词（直接回车表示不搜索）: 合同纠纷
最大翻页数（默认5）: 5
确认开始？(Y/n): y
```

```

    - 点击右上角"登录"按钮
    - 填写账号: 017656
    - 填写密码: Lhy740220
    - 提交登录

[步骤 2] 跳转北大法宝
    - 在图书馆页面找到"北大法宝"链接
    - 点击进入北大法宝

[步骤 3] 搜索案件
    - 输入搜索关键词
    - 执行搜索

[步骤 4] 收集URL
    - 遍历搜索结果列表
    - 翻页收集案件链接
    - 保存到 urls.txt

[步骤 5] 下载案件
    - 逐个访问案件页面
    - 提取标题和内容
    - 保存为txt文件到 downloads/ 目录
```

## 输出文件

- `downloads/` - 下载的案件文本文件
- `urls.txt` - 待下载的URL列表
- `debug_*.html` - 调试文件（出错时生成）

## 注意事项

1. **必须先启动Chrome调试模式**
2. **确保Chrome版本较新**
3. **保持网络畅通**
4. **不要关闭Chrome浏览器**

## 故障排查

### 连接浏览器失败
```
[✗] 连接失败
```
**解决**: 确保已运行 `启动Chrome调试模式.bat`

### 未找到登录按钮
```
[✗] 未找到登录按钮
```
**解决**: 检查Chrome是否正常打开了图书馆页面

### 登录失败
```
[✗] 登录失败，仍在登录页面
```
**解决**: 检查账号密码是否正确（账号: 017656, 密码: Lhy740220）
